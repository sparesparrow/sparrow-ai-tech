Automated Deployment Pipeline

A modern, secure CI/CD workflow using GitHub Actions to automate deployment to GitHub Pages, ensuring reliability and eliminating manual errors.

✓ .github/workflows/deploy.yml
  1. 1

    Trigger

    Workflow runs on push to `main` or manual dispatch.

  2. 2

    Build Job

    Checks out code, sets up Node.js v20, installs dependencies with `npm ci`, and runs `npm run build`.

  3. 3

    Upload Artifact

    Uses `actions/upload-pages-artifact` to upload the `dist` directory.

  4. 4

    Deploy Job

    Depends on successful build, then uses `actions/deploy-pages` to publish the artifact to GitHub Pages.

Final step: Configure the repository's GitHub Pages source setting to **"GitHub Actions"**.
