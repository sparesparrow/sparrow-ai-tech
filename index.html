Proč MCP? Proč tento stack?

Vývoj agentních AI systémů přináší nové výzvy – zejména tzv. "prompt rot", kdy se důležité prompty ztrácejí, nejsou verzované a brání efektivní spolupráci. MCP (Model Context Protocol) je otevřený standard, který řeší tuto fragmentaci a umožňuje interoperabilitu mezi nástroji, agenty a vývojáři.

  • Architektonické inovace: MutablePrompt umožňuje konverzi promptů mezi formáty (JSON, MDC, PGAI, šablony) bez zásahu do jádra serveru. Modularita a přepis do Rustu zajišťují výkon, bezpečnost a dlouhodobou udržitelnost.
  • Otevřené standardy a komunita: Všechny projekty jsou open-source (MIT licence), s důrazem na dokumentaci, kontejnerizaci a automatizované CI/CD. Aktivní zapojení do MCP komunity a viditelnost na specializovaných platformách.
  • Strategická doporučení: Důraz na stabilitu, urychlení Rust přepisu, formální prezentace toolchainu jako jednotného stacku a posílení komunitní angažovanosti.

MCP Ekosystém: Otevřený Stack pro Agentní AI

Naše open-source projekty tvoří ucelený ekosystém pro vývoj agentních AI systémů na bázi Model Context Protocolu (MCP). Každý nástroj řeší klíčovou výzvu v oblasti správy promptů, orchestrací projektů a návrhu workflow. Všechny projekty jsou dostupné na GitHubu a vítáme příspěvky komunity.

mcp-prompts

Server pro správu promptů a šablon s verzováním, více úložišti (souborový systém, PostgreSQL, MDC) a rozhraním MutablePrompt pro konverzi mezi formáty. Řeší problém "prompt rot" a je základním stavebním kamenem MLOps pro AI agenty.

GitHub

mcp-prompts-rs

Výkonný přepis mcp-prompts v jazyce Rust pro vyšší stabilitu, bezpečnost a škálovatelnost. Zaměřeno na produkční nasazení a budoucí rozšíření.

GitHub

mcp-project-orchestrator

Automatizuje zakládání nových MCP projektů, správu šablon a generování diagramů. Umožňuje agentům vytvářet projekty na základě požadavků uživatele.

GitHub

mcp-router

Návrhář workflow a router pro agentní systémy. Umožňuje navrhovat, monitorovat a řídit komplexní agentní workflow v rámci MCP ekosystému.

GitHub

mcp-prompts-catalog

Standardizovaný katalog promptů v JSON formátu, oddělený od serverové logiky. Umožňuje sdílení a verzování obsahu napříč implementacemi.

GitHub
